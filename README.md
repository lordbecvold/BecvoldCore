# Minecraft survival core plugin

# Components
* SetHome
* OnePlayer sleep
* Coordinates in hud
* Ram usage monitor
* Timer util (you can set day or night time longer then normal)
* User Info

## Dependencies
* Spigot
   * Version: **1.18.1**
   * [Website](https://getbukkit.org/)
