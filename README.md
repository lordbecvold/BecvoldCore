# Minecraft survival core plugin

> [!WARNING]
> ### End of support
> This project will no longer be updated or maintained.

# Components
* SetHome
* OnePlayer sleep
* Coordinates in hud
* Ram usage monitor
* Timer util (you can set day or night time longer then normal)
* User Info

## Dependencies
* Spigot
   * Version: **1.18.1**
   * [Website](https://getbukkit.org/)
